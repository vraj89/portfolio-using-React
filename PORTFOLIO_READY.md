## 🚀 Getting Started (3 Simple Steps)

### 1️⃣ Install Dependencies
```bash
npm install
```

### 2️⃣ Setup EmailJS (Optional but Recommended)
- Go to [emailjs.com](https://emailjs.com) - free account
- Create email service (2 minutes)
- Create template (1 minute)
- Copy 3 keys to `.env.local`
- Update email in `src/Contact.jsx`

See **QUICK_START.md** for detailed step-by-step instructions.

### 3️⃣ Run Your Portfolio
```bash
npm run dev
```
Visit: `http://localhost:5173` ✅
